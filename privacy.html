Privacy Policy

Explainish Chrome Extension  ·  Last updated: February 2026

Explainish is a Chrome extension that helps you summarize and understand web content using Google's Gemini AI. This policy explains what data is collected, how it is used, and what is never collected.

What we collect

Explainish collects anonymous usage analytics to understand how the extension is being used and to improve it over time. This includes:

  • A randomly generated anonymous installation ID (not linked to your identity)
  • Feature usage events (e.g. "user clicked Summarize", "user used Explain")
  • General website category of the page being used on (e.g. "news", "tech") — not the URL or page content
  • Session and retention signals (e.g. how often the extension is opened)

This data is collected via Google Analytics (Measurement Protocol) and is fully anonymous. It cannot be used to identify you.

What we never collect

  • Your name, email address, or any personally identifiable information
  • The content of pages you visit or summarize
  • Your full browsing history or specific URLs
  • Your Gemini API key
  • Any data that could identify you as an individual

Your API key

Explainish requires a Google Gemini API key to function. This key is stored exclusively in your browser's local storage on your own device. It is never transmitted to us, never stored on any server, and never shared with any third party. The only use of your API key is to make direct requests from your browser to Google's Gemini API.

Page content

When you use Explainish to summarize a page or explain highlighted text, that content is sent directly from your browser to Google's Gemini API using your own API key. Explainish has no server — content never passes through any infrastructure we control. Google's handling of API requests is governed by Google's API Terms of Service.

Data sharing

We do not sell, rent, or share your data with any third party. Anonymous analytics data is processed by Google Analytics solely to help us improve the extension.

Data retention

Anonymous analytics data is retained in Google Analytics for 14 months, after which it is automatically deleted. No other data is retained by us because no other data is collected.

Children's privacy

Explainish is not directed at children under the age of 13. We do not knowingly collect any information from children.

Changes to this policy

If we make material changes to this policy, we will update the date at the top of this page. Continued use of the extension after changes constitutes acceptance of the updated policy.

Contact

If you have any questions about this privacy policy, please open an issue on the extension's Chrome Web Store listing page or contact us via the support email listed there.

Explainish is an independent product and is not affiliated with Google LLC.
